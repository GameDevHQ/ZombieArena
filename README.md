# Zombie arena
The Zombie arena game, which is based on ideas and concepts from a "Beginning C++ Game Programming" book by John Horton.

Hotkeys
--------

- "W" / "S" / "A" / "D"  for movement

Requirements
------------
- C++11
- SFML 2.4.2 (x32 version)
